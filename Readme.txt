For Amazon Rekognition:


The AMR_bucket.py file reads the image from the amazon s3 bucket and outputs the label and confidence of the specified image in the form of bar graph.                                 
The AMR_local.py file reads the image from the local directory and outputs the label and confidence of the specified image in the form of bar graph.                               
The new_user_credentials.csv file contains the credentials to access the console.            
The labels.pdf file contains the image and labels of the object that was tested in amazon API With different epsilon value such as: 0.01,0.1 and 0.5. It shows the confidence percent on the Y-axis and the label detected in the X-axis.                
The IMG243.pdf displays the confidence percentage and label detected in the text format.
